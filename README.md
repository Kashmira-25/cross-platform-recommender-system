# Cross-Platform Recommender System

## Overview

This project aims to build a system that integrates user activity from multiple platforms (YouTube, Shopping, Learning apps) and generates personalized recommendations based on unified user behavior.

## Features

* Multi-platform data integration
* Unified user interest profiling
* Cross-platform recommendations
* Interest drift detection
* Feedback-based learning

## Tech Stack

* Python
* pandas, NumPy
* scikit-learn
* Streamlit
